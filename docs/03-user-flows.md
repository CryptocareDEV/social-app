# User Flows

## New User
1. Lands on app
2. Signs up
3. Enters feed
4. Creates first post
5. Sees post in feed and profile

## Returning User
1. Logs in
2. Scrolls feed
3. Likes a post
4. Downloads a post (watermark applied)
5. Creates new post

## Creator Flow
1. Click "Create"
2. Choose type (text/image/video/meme)
3. Publish
4. See post instantly in feed and sidebar

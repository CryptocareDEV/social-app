# MVP Scope (Phase 1 Lock)

### Users can:
- Sign up and log in
- Create posts:
  - Text
  - Image
  - Video
  - Meme (basic editor)
- View a global feed
- View their own profile
- View a list of their posts (sidebar)
- Like posts
- Download images/videos with a watermark

### Users cannot:
- Comment
- Follow
- Message
- Edit posts
- See algorithms or recommendations
